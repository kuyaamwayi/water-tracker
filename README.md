# Water Tracker

NFC-triggered water intake tracker with a home screen widget.

## How it works
1. Tap your NFC tag → logs 500ml instantly
2. Widget on home screen shows daily progress + streak
3. Resets at midnight, tracks streaks

## NFC Tag Setup
Write this URL to your NFC tag using **NFC Tools** app:
```
watertracker://log-bottle
```

## Download APK
Go to **Actions** tab → latest build → **water-tracker-debug** artifact → download and install.

Enable "Install from unknown sources" on your phone first:
Settings → Apps → Special app access → Install unknown apps → Files → Allow
